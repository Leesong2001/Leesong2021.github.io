56.合并区间

以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi] 。请你合并所有重叠的区间，并返回 一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间 。

提示：

  • 1 <= intervals.length <= 104
  • intervals[i].length == 2
  • 0 <= starti <= endi <= 104

解题思路

这段代码的解题思路是合并重叠的区间。给定一个包含多个区间的二维数组 intervals，每个区间表示为 [start, end]，其中 start 表示区间的起始位置，end 表示区间的结束位置。需要将重叠的区间合并，并返回合并后的区间结果。

解题思路如下：

  1. 定义一个结构体 interval，用来表示区间的起始位置和结束位置。
  2. 定义排序规则的比较函数 compareInterval，用于对区间进行排序。排序规则是先按照起始位置升序排序，如果起始位置相同，则按照结束位置升序排序。
  3. 创建一个空的 vector 容器 intervals_sorted，用于存储排序后的区间。
  4. 遍历给定的区间数组 intervals，将每个区间转换为 interval 结构体，并添加到 intervals_sorted 中。
  5. 使用 std::sort 函数对 intervals_sorted 进行排序，排序规则为使用上述定义的比较函数 compareInterval。
  6. 创建一个空的 vector 容器 answer，用于存储合并后的区间结果。
  7. 创建一个临时的 vector 容器 itv_meg，用于存储当前合并的区间的起始位置和结束位置。
  8. 遍历排序后的区间 intervals_sorted。
    • 若是第一个区间，直接将其起始位置和结束位置赋值给 itv_meg。
    • 若存在交集（当前区间的起始位置小于等于当前合并区间的结束位置），则更新合并区间的结束位置为当前区间的结束位置。
    • 若不存在交集，将当前合并区间添加到结果 answer 中，并更新合并区间为当前区间。
  9. 将最后一个合并区间 itv_meg 添加到结果 answer 中。
  10. 返回合并后的区间结果 answer。

通过以上步骤，代码能够将重叠的区间进行合并，并返回合并后的区间结果。

时间复杂度分析

  • 对给定的区间进行转换，将每个区间表示为 interval 结构体，需要遍历所有区间，时间复杂度为 O(n)，其中 n 是区间的数量。
  • 对转换后的区间进行排序，使用了 C++ 的 std::sort 函数，其时间复杂度为 O(n log n)，其中 n 是区间的数量。
  • 遍历排序后的区间，进行合并操作，需要遍历所有区间，时间复杂度为 O(n)，其中 n 是区间的数量。

综上所述，整体的时间复杂度为 O(n log n)。

空间复杂度分析

  • 创建了一个临时的 vector 容器 intervals_sorted，用于存储排序后的区间，其空间复杂度为 O(n)，其中 n 是区间的数量。
  • 创建了一个临时的 vector 容器 answer，用于存储合并后的区间结果，其空间复杂度为 O(n)，其中 n 是区间的数量。
  • 创建了一个临时的 vector 容器 itv_meg，用于存储当前合并的区间的起始位置和结束位置，其空间复杂度为 O(1)。

综上所述，整体的空间复杂度为 O(n)。
