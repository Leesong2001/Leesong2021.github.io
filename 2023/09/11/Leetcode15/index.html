15.三数之和

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。

请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

提示：

  • 3 <= nums.length <= 3000
  • -105 <= nums[i] <= 105

解题思路

  1. 初步：三重循环遍历nums[i,j,k]，判断是否和为0，复杂度为$O(n^3)$
  2. 优化：二重循环遍历nums[i]与num[j]，计算得到sum=nums[i]+num[j]，则第三个元素必须为0-nums[i]-nums[j]
  3. 难点：如何去重，保证答案中不可以包含重复的三元组。

算法步骤

  1. 输入nums数组，对nums数组进行排序，得到nums_sorted
  2. 然后遍历nums_sorted数组并对各元素计数，得到nums_unique和nums_unique_quantity。其中nums_unique按升序存储nums_sorted的不重复元素，等效于set(sort(nums))，而nums_unique_quantity则为nums_unique各位置元素在原始nums数组中的数目。
  3. 遍历取数，双指针+二分优化
    1. 指针i遍历nums_unique的每一个位置，并对nums_unique_quantity[i]减1，相当于取了一个数。
    2. 指针j从指针i当前位置开始判断，若nums_unique_quantity[i]>=1，则取当前位置，否则j++;
    3. 取完nums_unique[i]与nums_unique[j]后，先确定二分搜索的起始位置k，与第二步同理，k=j或者j+1，然后对nums_unique数组在<k,end>区间进行二分搜索，判断是否能找到0-nums_unique[i]-nums_unique[j]。若能，则将三元组添加进答案。
